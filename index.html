Token oluşturmak için: GitHub → Settings → Developer settings → Personal access tokens → Fine-grained token oluşturun. Repo seçin: "Contents: Read and write" izni verin. Token ghp_ (küçük harf) ile başlamalı.

Fotoğraf

İrsaliye Seç
Fotoğraf Çek / Ekle
QR için Fotoğraf Linki (Bulut)
Atölye QR'ı taradığında fotoğrafı internetten yüklemek için link gerekli. Fotoğrafı OneDrive / Google Drive / Dropbox / WhatsApp'a yükleyip paylaşım linkini buraya yapıştırın. Otomatik yükleme denendi ama başarısız oldu veya şirket sırrı içeren fotoğraflar için bu daha güvenli.
📖 Hızlı rehber:
OneDrive: Fotoğrafa sağ tıkla → Paylaş → Link kopyala
Google Drive: Fotoğrafa sağ tıkla → "Linki al" → "Linkin olan herkes görebilir" seç
WhatsApp Web: Fotoğrafı kendine gönder → Açıldığında URL'yi kopyala (geçici)
Link yapıştırınca otomatik olarak uygun formata çevrilir.
